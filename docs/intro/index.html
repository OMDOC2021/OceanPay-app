    通用说明

    Ocean Pay App接入流程：

    1. 请求测试环境地址 https://test-payway.cathayinter.com/
    2. 账号均为web端注册的账户,所以 账号不需要注册
    3. 所有的请求均需要携带header x-app 固定为OceanPay
    4. 请求分为GET, POST
    5. POST 请求 Content-Type: application/json
    6. 需要接入FCM, 需要实现前后台接收消息
