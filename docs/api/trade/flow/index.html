    交易流水

    请求URL#

    /app/trade/list

    请求方式#

    GET请求

    请求Header#

    字段类型长度示例说明
    x-appstring8"OceanPay"固定为OceanPay
    x-tokenstring64"fc41dad58a764e7987c26d103e08d610"登录后token

    请求字段#

    字段类型长度示例说明
    timestampLong13"1639016573375"时间戳,系统会根据时间戳往前推5天交易

    请求示例#

    GET /app/trade/list?timestamp=1639016573375 HTTP/1.1Host: https://test-payway.cathayiner.comx-app: OceanPayx-token: 32aee8b4d9ac428e8c5fa20b1eadd39bCookie: JSESSIONID=f0dc6e68-1196-44f9-bc72-1dffe2ff3785

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否成功
    msgstring---"ok"否说明
    ---------
    nextTimeStamplong13否下一回请求时间戳
    tradesMap---是按日返回交易数据,以及统计值
    ---------
    totalint---否当日交易总金额
    recordsObject---
    finishedTimeString---2021-11-05 10:52:08否交易时间
    typeString---"ktbcard"否支付类型
    amountint---1否交易金额 1 标示 1 THB
    tradeNumberString---"1456468850848694272"否交易单号

    返回内容示例#
