    经营报表

    请求URL#

    /app/trade/statistics

    请求方式#

    GET请求

    请求Header#

    字段类型长度示例说明
    x-appstring8"OceanPay"固定为OceanPay
    x-tokenstring64"fc41dad58a764e7987c26d103e08d610"登录后token

    请求字段#

    字段类型长度示例说明
    typestring3-5"day"报表统计类型 day, week, month
    startDatestring8-32"2021-11-12"传当日日期即可,系统会根据类型按日,按周,按月统计

    请求示例#

    GET /app/trade/statistics?type=day&startDate=2021-11-10 HTTP/1.1Host: https://test-payway.cathayiner.comx-token: 32aee8b4d9ac428e8c5fa20b1eadd39bx-app: OceanPayCookie: JSESSIONID=f0dc6e68-1196-44f9-bc72-1dffe2ff3785

    请求示例中 日期说明#

    startDatetype为 `day` 时, 请求需要传当日日期 2021-12-09type为 `week` 时, 请求需要传当日日期即可 2021-12-09type为 `month` 时, 请求需要传当日日期即可 2021-12-09

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否成功
    msgstring---"ok"否说明
    ---------
    numberstring---"0.00"否总交易笔数
    perOrderstring---"0"否单笔均价 1 标示 0.01 THB，以分为单位
    totalstring---"0.00"否总交易金额 1 标示 0.01 THB，以分为单位

    返回内容示例#

    {  "msg": "操作成功",  "code": 200,  "data": {    "perOrder": "1",    "number": "2",    "total": "2"  }}
