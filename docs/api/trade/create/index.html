    创建订单

    请求URL#

    /app/trade/create

    请求方式#

    POST请求，Content-Type: application/json

    请求Header#

    字段类型长度示例说明
    x-appstring8"OceanPay"固定为OceanPay
    x-tokenstring64"fc41dad58a764e7987c26d103e08d610"登录后token

    请求字段#

    字段类型长度示例说明
    typestring---"promptpay"付款类型,promptpay
    amountint--1001THB = 100, 以分为单位
    timestamplong19"1639152000000"订单时间戳
    remarkstring---"测试用"备注

    请求示例#

    {    "type":"promptpay",    "amount":100,    "timestamp":1635696000000,    "remark":""}

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否成功
    msgstring---"ok"否说明
    ---------
    createTimeString---"2021-11-05 10:50:08"否交易发生时间
    qrCodeString---ktbcard否base64二维码
    qrCodeExpireSecString---600否二维码有效期(秒)
    payChannelString---OP否通道标识
    paymentMethodString---promptpay否支付类型
    tradeNumberString---1456468850848694272否交易单号
    merchantTradeNumberString---"112233445123"否商户交易单号
    remarkString---"Code No. 1636084208574"否备注
    statusString---"PAYING"否等待支付
