    我的结算信息

    请求URL#

    /app/settle/info

    请求方式#

    GET请求

    请求Header#

    字段类型长度示例说明
    x-appstring8"OceanPay"固定为OceanPay
    x-tokenstring64"fc41dad58a764e7987c26d103e08d610"登录后token

    请求示例#

    GET /app/settle/info HTTP/1.1Host: https://test-payway.cathayiner.comx-app: OceanPayx-token: fc41dad58a764e7987c26d103e08d610Cookie: JSESSIONID=1bf432bf-8870-4756-bf18-63bf0e2aaa5a

    返回内容说明#

    字段类型长度示例可否为空说明
    codeint---200否成功
    msgstring---"ok"否说明
    ---------
    settleBankNamestring---"工商银行"否银行名称
    settleBankAccountstring---"11223344556677"否银行账号
    settleNamestring---"kuuk"否账户姓名

    返回内容示例#

    {  "msg": "操作成功",  "code": 200,  "data": {    "settleBankName": "工商银行",    "settleBankAccount": "11223344556677",    "settleName": "kuuk"  }}
